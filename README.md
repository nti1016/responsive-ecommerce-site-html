#  Multi-Page E-commerce Website

This is a front-end e-commerce website project built using HTML. The project demonstrates how multiple web pages can be structured and linked together to function as a complete system.

## Features

* Multi-page website structure
* Fully linked navigation between pages
* Responsive sign-in interface
* Clean and simple user interface design
* Beginner-friendly code structure

## Pages Included

* Home Page
* Product Page
* Sign-In / Login Page
* Navigation Links between all pages

##  Technologies Used

* HTML5
* css

##  Project Goal

The goal of this project is to understand how to:

* Structure a real-world website
* Link multiple pages together
* Build a simple and functional user interface

## Future Improvements

* Add CSS for better styling and responsiveness
* Integrate JavaScript for interactivity (cart, validation, etc.)
* Implement backend functionality for authentication and payments
